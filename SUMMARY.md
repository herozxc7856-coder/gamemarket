## 🗄️ База данных для продакшена:

**SQLite не подходит для Vercel** (файлы удаляются после деплоя).

### Используйте PostgreSQL:

1. **Бесплатно**: [Neon.tech](https://neon.tech) или [Supabase](https://supabase.com)
2. Создайте БД → получите `DATABASE_URL`
3. Обновите `prisma/schema.prisma`:
   ```prisma
   datasource db {
     provider = "postgresql"
     url      = env("DATABASE_URL")
   }
   ```
4. Добавьте `DATABASE_URL` в **Vercel → Settings → Environment Variables**
