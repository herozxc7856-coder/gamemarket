## 🚀 Установка минимальной версии

```cmd
# 1. Откройте терминал ОТ ИМЕНИ АДМИНИСТРАТОРА

# 2. Перейдите в папку:
cd C:\Users\genius\Downloads\gamemarket

# 3. Создайте резервную копию оригинального package.json:
copy package.json package.json.full

# 4. Замените на минимальный:
copy package.minimal.json package.json

# 5. Настройте зеркало:
npm config set registry https://registry.npmmirror.com

# 6. Очистите кэш и старые файлы:
npm cache clean --force
rmdir /s /q node_modules 2>nul
del package-lock.json 2>nul

# 7. Установите минимальные зависимости:
npm install --legacy-peer-deps --timeout=300000

# 8. Запустите:
npm run dev

# 9. Откройте:
# 👉 http://localhost:3000
```

## 🔧 Как вернуть полную версию:

```cmd
# 1. Остановите сервер (Ctrl+C)

# 2. Восстановите оригинальный package.json:
del package.json
copy package.json.full package.json

# 3. Установите полную версию (когда сеть будет работать):
install-ru.bat
```

## 🎯 Тестирование без бэкенда:

1. Запустите минимальную версию
2. Откройте `http://localhost:3000/sell`
3. Заполните форму — увидите расчёт комиссии
4. Нажмите "Опубликовать" — появится сообщение об успехе
5. **Данные не сохранятся** (нет БД), но интерфейс работает!

## 🆘 Нужна полная версия?

Когда сетевые проблемы решатся:

1. Восстановите `package.json` (см. выше)
2. Запустите `install-ru.bat`
3. Выполните `npx prisma db push`
4. Запустите `npm run dev`

И всё заработает с базой данных! 🗄️✨
